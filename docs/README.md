# Cascade CSS Documentation

Welcome to the Cascade CSS documentation. This directory contains all the guides and references you need to work with the framework.

> **Important Notice**: This is an independent CSS framework that is NOT affiliated with, endorsed by, or officially related to Automatic.css. This is a separate implementation with its own architecture.
